## Description
Sammy er en agent for [[02 Player/Erukana (Nissen)/People/Fafnir\|Fafnir]]s modstandsgruppe. Hun fungerer som kontaktperson og talsmand for gruppen, og hjælper med at koordinere operationer mod [[02 Player/Erukana (Nissen)/Factions/Blodrosen\|Blodrosen]].
